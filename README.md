# Express基础MVC目录结构
初始化目录结构，新项目的模板。

## 相关信息
* 前端CSS语言使用 LESS
* Express MVC框架
* EJS 模板引擎
* Node.js 语言
* Grunt 作为自动化工具
* Jasmine 单元测试
* 版本控制：Github
* MongoDB+Mongoose

## 版本信息
* body-parser （1.13.2）
* cookie-parser （1.3.5）
* debug （2.2.0）
* ejs （2.3.3）
* express （4.13.1）
* less-middleware （1.0.x）
* morgan （1.6.1）
* serve-favicon （2.3.0）
